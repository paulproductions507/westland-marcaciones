📧 Prueba manual con fechas personalizadas
✅ Cómo configurar para que el correo llegue con la tabla (3 pasos):

1) emailjs.com → Add Service → conecta Gmail o Microsoft 365 → copia el Service ID
2) Email Templates → Create New Template → En el template:
  • Subject: {{subject}}
  • Content (Body): Selecciona modo HTML y escribe solo:
    {{{html_content}}}
  • To Email: {{to_email}}
  Guarda y copia el Template ID
3) emailjs.com → Account → General → copia la Public Key

⚠️ Importante: El campo Body/Content DEBE estar en modo HTML (no texto). Si solo ves texto plano, busca el botón "Source Code" o "</>" en el editor del template.
🔗 Para Power Automate (Office 365):
Crea un flujo en Power Automate con trigger "Recurrence" (ej. viernes 6:30pm) → acción "HTTP" GET a:
https://westland-marcaciones-default-rtdb.firebaseio.com/historial_v1.json → acción "Send an email (V2)" con Outlook → en el campo Body activa "Is HTML" → pega el reporte. Así usas tu firma de Office 365 directamente.
